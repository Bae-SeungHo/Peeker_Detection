
파일/폴더 설명
----------------
Add-on				학습된 인공지능 모델 저장 폴더
Dataset				프로그램 실행에 필요한 사진 데이터 폴더 (학습에 사용된 연령별 한국인 데이터셋은 미첨부)
Videos				프로그램 작동 검증에 사용된 샘플 비디오 폴더
Web				웹으로 구현된 STM기 화면의 연령별 웹 파일 폴더

KB_Mask_Face_Detector.ipynb	사진에서 얼굴을 인식하여 인공지능 학습 전 전처리하는 코드
KB_Mask_Learning.ipynb		마스크 착용 얼굴을 학습하는 코드
KB_Main.ipynb			인공지능 모델을 통하여 구현된 코드
